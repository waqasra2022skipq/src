hitAppoint Help

Thank you for choosing hitAppoint! hitAppoint is online appointment scheduling platform for small businesses and solo practitioners.

Installation

  1. Download the latest version zip file.
  2. Extract all the files from this zip archive on your computer.
  3. Create a new MySQL database at your website for hitAppoint. If you already have a database, and you can't or don't want to create another one specifically for hitAppoint, hitAppoint can easily make use of this existing database. The names for database tables starts with hitAppoint's own prefix thus they do not interfere with other applications.
  4. Rename the db.rename_it.php to db.php, then open this new db.php file in your text editor and enter your MySQL database login information. The file already contains some sample values so it will be easy.
  5. Decide in which directory you will be installing hitAppoint on your website. You would probably want to create a new directory. Upload all the files keeping their directory structure there.
  6. Point your browser to the web page where you have uploaded the files. You will be transferred to the setup screen. Please follow the instructions on the screen.
  7. Next please review the Quick Start Guide part at www.hitappoint.com.
